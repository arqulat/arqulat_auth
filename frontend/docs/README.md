## Quick Start

```bash
# 1. Navigate to frontend directory
cd arqulat_auth/frontend

# 2. Install dependencies
npm install

# 3. Run development server
npm run dev
```

The app will be available on the default Vite port (e.g., `http://localhost:5173`).
